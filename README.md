# Bmw-price-prediction-and-recomendations
BMW Price Prediction &amp; Recommendation System using Machine Learning 
This project analyzes BMW car data and predicts prices using Python.

## Files
- BMW_PRICE_PREDICTION.ipynb → Jupyter Notebook
- bmw_50k_dataset.csv → Dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## How to Run
1. Open the notebook in Jupyter
2. Run all cells step by step
